What is a Weather App?

The Open Weather Map provides free weather data across the globe. Our Weather App is a webpage that can …

  1. interface with Open Weather Map API
  2. make AJAX requests
  3. fetch weather data from the server
  4. display weather data

What is API & AJAX?

Our webpage can communicate with Open Weather APIs (Application Programming Interface) by AJAX (Asynchronous JavaScript And XML, formally known as XMLHttpRequest Object, or XHR).

In short; AJAX is about loading data in the background and display it on the webpage, without reloading the whole page. –W3Schools

AJAX request gets data, often in JSON rather XML format, from the server, and the JavaScript dynamically change the HTML code to display the data. API & AJAX is common method with which Twitter loads more tweets and Google Map display more details by dynamic JavaScript without reloading

Under default browser Same Origin Policy, a web page can only make AJAX requests to the same domain it was loaded from. For example, after I log in my ebanking account in my browser, I can make AJAX requests that are trusted by my bank; if I visit a malicious website which sends AJAX requests to transfer money out of my bank account, my browser will block all AJAX requests because I have not authenticated by logging in my account. This is a general security measure.

However, there are some exceptions. CORS (Cross-Origin Resource Sharing) standard is implemented by modern servers and browsers. This enables APIs specifying which domains are allowed to make AJAX requests. Some APIs allow all domains to make AJAX requests, such as Open Weather Map API Open Weather Map. It permits cross-origin requests and makes its weather data available to any website, from any domain. After all, this is only weather data, not bank saving.

How to make AJAX requests by jQuery?

jQuery Method Code
load() Method $(selector).load(URL,data,callback);
$.get() Method $.get(URL,callback);
$.post() Method $.post(URL,data,callback);

These jQuery Methods often used in combo with $("button").click(function(){}); and if (failedLoading) alert ("XXX")

1. jQuery load() Method

  1. load data from a server
  2. put the returned data into the selected element.
<script>
$(document).ready(function(){
    $("button").click(function(){
        $("#div1").load("demo_test.txt", function(responseTxt, statusTxt, xhr){
            if(statusTxt == "success")
                alert("External content loaded successfully!");
            if(statusTxt == "error")
                alert("Error: " + xhr.status + ": " + xhr.statusText);
        });
    });
});
</script>
