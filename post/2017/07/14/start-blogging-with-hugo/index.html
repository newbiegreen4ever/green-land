Start Blogging with Hugo

Why blogging?

There are several good reasons to blog:

  1. Build my memory palace

A personal blog enables you to organize content in a fully customized fashion. This makes it easier to collect and retrieve information for later use.

  1. Express myself with clarity

Writing is the best way to capture diffusing thoughts and fleeting inspirations.

  1. Play with self-hosting

Self-hosting sounds cool.

I. Get started with Jekyll

A friend mentioned Jekyll when talking about building and hosting sites. Being naively stupid, I started googling and reading about Jekyll right away without giving much thought on other alternatives. This mistake cost me nearly 8 hours of leisure time.

The official Jekyll on Windows walkthrough is my main source of information. I installed Jekyll via Chocolatey, a package manager on Windows. Before I could run Jekyll on Window, I had to install Ruby, the engine that rugns Jekyll.
