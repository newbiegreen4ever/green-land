Summary

This article adopts a specific-to-abstract 4-stage approach to explain array functions. A quasi-map function is developed from multiplying each items in arr by 3. 1. for-loop 2. Multiply-by-three function 3. Multiply-by-anything function 4. Map-transformation function

Note that the callback for map() can take 3 types of arguments from an array I. item II. index III. array itself

I write my own examples referring to articles in the reading list. All examples have run successfully in Google Developer console.

I. Map() callback taking item parameter

var arr = [1, 3, 5, 7, 9];

function squareItem (item) {
    return item*item;
}

function map (arr, transformFun) {
    var arrMapped = [];
    for (var i =0; i < arr.length; i++) {
        arrMapped[i] = transformFun( arr[i] );
    }
    return arrMapped;
}

map (arr, squareItem);

II. Map() callback taking index parameter

var arr = ["apple", "banana", "cat", "dog", "egg", "fish"];

function listItem (arr, item) {
    index = arr.indexOf(item) + 1;
    return index + ". " + item;
}

listItem (arr, "apple");

function map (arr, transformFun) {
    var arrMapped = [];
    for (var i =0; i < arr.length; i++) {
        arrMapped[i] = transformFun( arr, arr[i] );
    }
    return arrMapped;
}

map (arr, listItem);

III. Map() callback function taking array itself as parameter

var myArray = [1,2,3];

function arrayCounter (array) {
    if (array instanceof Array) {
        return array.length;
    } else {
        return 0;
    }
}

myArray.map(arrayCounter);

Format

I can directly write a function inside the map call. Ah ha ~ Finally I figure out how this weird format works!

var arr = [1, 2, 3, 4, 5];
var arrTimesThree = arr.map(function(item) {
   return item * 3;
});
console.log(arrTimesThree); // -> [3, 6, 9, 12, 15]
